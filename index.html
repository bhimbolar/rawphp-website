The PHP Framework for everyone!

RawPHP is a super-easy and super-light PHP framework for people from different PHP backgrounds such as laravel, cakephp, slim and procedural PHP, working together on a single project!

  • You can comfortably use your Cakephp and Laravel ORM, packages and middlewares inside it and write the code of any framework you are comfortable with.
  • It is also the ideal framework for building robust API/webservice platforms.
  • Its the right framework for people learning a PHP framework for the first time because of its easy to learn, elegant and powerful syntax.
